Types of List Elements

There are two types of list elements in HTML namely:

Ordered List

This is a numbered list. It uses the tag ol e.g.

  1. Wake up at 4:00 am
  2. Learn how to code/design for an hour
  3. Go for an hour run

Unordered list

This is a bullet list. It is denoted by the tag ul e.g.

  • Weetabix
  • Soap
  • Cooking oil
