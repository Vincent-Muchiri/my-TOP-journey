Links

Links in websites are defined by the HTML anchor tag a. The name of the link is written with the anchor tag and the actual link is defined by the attribute 'href'.

There are two types of links:

Absolute links

Absolute links connect to web pages outside the website. They have follow the format protocol://domain/path. An example is TOP About Page

Relative links

A relative link connects to a different section of the webpage you are already in or a different webpage in the same website you are in. They DON'T have the protocol or domain section, just the path. An example of this is our own website's About page

Images

Images in HTML are embedded using the element tag img. The crucial attributes for images are:

  • src- Defines the location of the image as either an absolute or relative link.
  • alt- Text that describes the image incase the image fails to load and for the visually impaired.
